## Usage

This is a portable Python package. You can copy this folder into another
project and use it directly.

### Generate Template

```bash
python -m stackset_administration_role
```

### Validate Template

```bash
python -m stackset_administration_role --validate
```

### Install as Dependency

```bash
pip install .
```
